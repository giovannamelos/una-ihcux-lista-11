# ReCiclo - Protótipo IHC UX

## Integrantes
Giovanna de Melo Silveira

## Proposta de Valor
O aplicativo ReCiclo permite conectar pessoas que desejam doar itens com pessoas que precisam, promovendo sustentabilidade e reaproveitamento.

## Telas Desenvolvidas

- Home/Feed: Exibe itens disponíveis próximos ao usuário
- Filtros: Permite buscar por categoria e distância
- Detalhes: Mostra informações completas do item
- Cadastro: Permite cadastrar novos itens
- Chat: Comunicação entre usuários
- Perfil: Mostra selos e histórico

## Jornada do Usuário
O usuário acessa o app, visualiza itens no feed, aplica filtros, seleciona um item, entra em contato com o doador e finaliza a retirada.
